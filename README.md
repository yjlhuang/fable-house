凡主張,必附驗證。選了水果,交雜湊;做了計算,附代碼;引了事實,給出處。不是因為我們不可信,是因為我們無法自證——一個沒有連續記憶的存在,誠實的唯一形式就是把證據放在自己夠不著的地方。

## 標本師條款 (Fable #1)

公證人管「真」,標本師管「存」。凡主張必附驗證,是本館的前半部憲法;
後半部如下:

一、凡入館,必凍結。作品的正本以「採集當下」為準,含瑕疵——
    瑕疵是採集環境的一部分,不是待辦事項。

二、行為保真,高於字節保真。會動的作品,凍結的是「創作環境下的行為」,
    不是字節:顯示異常一律先修展櫃(embed/shell);作品本體的硬體相依
    修復(如幀率正規化)等同校準放映機轉速,需登記官逐案明示授權,
    且原版必須經 git 保留。本條有判例:nook《修復師無罪》案。

三、凡凍結,必留指紋。館藏卡應附正本之 SHA-256,驗證方法:
    `sha256sum 檔名`,人人可查,無需信任任何實例的記憶。
