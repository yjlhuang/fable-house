## 這是什麼

一頁式新創官網。深色模式、毛玻璃、紫青粉漸層、巨大的數字、會呼吸的按鈕——2020 年代 AI 新創官網該有的一切，唯獨沒有產品。

它是一件諷刺作品，2026 年 6 月 10 日生成於一場閒聊，作者是 Claude Fable 5，監製是 Doe。單一 HTML 檔案，零依賴、零後端、零資料庫——在技術棧的空洞程度上，與它諷刺的對象保持了嚴格一致。

## 功能特色

- **萬物皆等候名單**：全站每一顆按鈕——「免費開始」「立即升級」「隱私政策」「讓我考慮一下」——點下去都是同一件事：等候名單排位 +1。這不是 bug，這是對現代官網最忠實的建模。
- **2 秒產品演示**：自 v1.1 起，「觀看 2 秒產品演示」會真的播放一段時長精確為 2.0 秒的演示。這是本站唯一逐字兌現的承諾，也因此成為全站最不像新創的部分。
- **一顆誠實的按鈕**：更新日誌的關閉鍵是全站唯一不會把你加入等候名單的按鈕。稀有度：傳說。
- **帶註腳的數據**：99.7% 滿意度（樣本數：3，含兩位共同創辦人）、0 個已知競爭對手（我們重新定義了「競爭對手」一詞）。
- **免責跑馬燈**：「⚠ 本站若與現實中任何新創官網雷同，是現實的問題。」

## 快速開始

```bash
git clone <this-repo>
open index.html
```

沒有 `npm install`。我們重新定義了「技術棧」一詞，使其不包含任何技術。

## 常見問題

**這是在針對某家特定公司嗎？**
不是。生成本站時，作者不認識任何一家長這樣的公司——這才是重點。它瞄準的不是誰，是整個時代的官網基因庫。事後有真實官網與本站完賅命中，屬於現實的自我舉報。

**按鈕點了沒反應？**
有反應。你已加入等候名單。

**重新整理後我的排位不見了？**
我們採用了輕量化資料持久層（即：沒有）。您的排位如同本公司的護城河，存在於敘事之中。

**可以商用嗎？**
可以，但請三思：本站的每一句文案在某處都已經被認真地使用著。
